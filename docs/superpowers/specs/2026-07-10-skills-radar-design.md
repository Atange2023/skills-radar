# 中文 Skills 专家推荐雷达

## 目标

把 `skills_guide.html` 中的专家、项目、场景和安装建议整理成一个可持续维护的中文互联网 Skills 推荐站。目标用户是中文 AI 开发者、内容创作者、研究者和知识工作者。

## 体验设计

- 首屏直接呈现今日状态、站点定位和“按场景开始”的入口。
- 用六个工作场景替代泛化的项目列表：开发、内容、调研、知识蒸馏、办公、多模态。
- 保留附件中的专家分组，并将每位专家的定位、适用人群和推荐顺序放在筛选结果上方。
- 支持专家、标签、优先级和关键词组合筛选；卡片显示用途、适合场景、风险提示和 GitHub 入口。
- 通过“今日复核”区域展示最近一次检查时间、来源数量和有变化的项目，形成内容可信度信号。

## 视觉方向

采用“编辑部 / 雷达”风格：浅纸色背景、深色排版、绿色表示稳定推荐、珊瑚色表示新鲜度、蓝色表示研究型项目。页面避免装饰性渐变和大面积浮夸插画，让密集内容易于扫描。

## 技术边界

- 使用静态结构化内容文件作为单一数据源，前端在浏览器内完成搜索和筛选。
- 初版不引入登录、数据库或用户投稿，降低发布和维护成本。
- 每天北京时间 00:05 运行复核任务，检查附件中 GitHub 来源的仓库状态、README 和推荐价值。
- 只有确认发生有意义变化时才更新内容、更新变更日志并重新发布；无法确认时保留现有内容并记录跳过原因。

## 验收标准

- 桌面和移动端均可浏览，首屏可直接进入场景或搜索。
- 26 个附件项目和 4 位专家均可检索，复制链接功能可用。
- 页面不依赖实时第三方 API 才能显示核心内容。
- 构建成功，自动复核任务使用 `Asia/Shanghai` 时区并覆盖内容更新与重新发布。
